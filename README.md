# AGE CALCULATOR AND CLASSIFICATION- introdution:
This is a Pyhton program that calculates your age and classifies you as an **adult** or  **underage**.
It also shows you if there is a **value error**, for example, if the result is negative.

# HOW TO USE IT:
1. Open the file Age_classification 
2. You have to **run** the file.
3. Once you are in the terminal, the script it´s going to guide you:
- First it welcomes you, then it asks you to:
    - Type the **current year**,
    - Type your **day of birth**.
You ONLY have to type numbers.
4. It will show you the result of the substration of the years, which gives your **age**.
5. Then it will show you in what **age classify** you are into.

# FEATURES
    - Age calculator
    - Age classification
    - Validations
